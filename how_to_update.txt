HOW TO UPDATE
*************

Terminal: cd /Users/SarahRobinson/Documents/GitHub/robinsonsarah23.github.io

Terminal: bundle exec jekyll serve

Make changes to .md

Preview at http://localhost:4000

Terminal: control-C

Update GitHub and wait a few minutes


*************
STYLE
*************

Original dinky code can be viewed at ./_sass/jekyll-theme-dinky.scss in TextEdit

Update sidebar in ./_layouts/default.html
